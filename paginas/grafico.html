Site de satisfação de cliente e gráficos

Um sistema de satisfação de clientes, com dois tipos de acessos,
um para clientes (satisfação) e o outro para administrador (gráfico).

Tecnologias usadas: Python, Django, Base de Dados, Docker, Javascript, Chart.js

Aprofundamento do sistema

Acesso às Avaliações das Lojas (URL: /avaliacao):
Nesta seção do sistema, os clientes têm a oportunidade de fornecer feedback
sobre o atendimento que receberam na loja. Cada loja possui seu próprio login
e senha, garantindo que as avaliações sejam específicas para cada estabelecimento.
Os clientes podem classificar sua experiência de 1 (péssimo) a 5 (ótimo),
permitindo uma avaliação detalhada e honesta dos serviços prestados.

Acesso Administrativo aos Gráficos (URL: /grafico):
Para os administradores das lojas, disponibilizei uma seção dedicada para visualização
de gráficos gerados dinamicamente com base nas avaliações dos clientes. Utilizando
o framework Chart.js, esses gráficos fornecem uma representação visual clara das avaliações por
loja e nota, permitindo uma análise rápida e eficiente do desempenho.

Painel Administrativo Completo (URL: /admin):
Por fim, desenvolvi um painel administrativo abrangente para os usuários com permissões
de administrador. Nesta seção, os administradores têm controle total sobre o sistema,
incluindo a gestão de lojas, avaliações, usuários e permissões de acesso.
Isso permite uma gestão eficaz e personalizada das operações, garantindo que as
necessidades de cada loja sejam atendidas de forma adequada.

Para implementar este sistema, utilizei Django com Python para o backend, e para o frontend,
fiz uso de HTML, CSS e JavaScript, aproveitando o poder do framework Chart.js para
a geração de gráficos dinâmicos. Para o deploy da aplicação, optei por utilizar
Docker para criar a imagem da aplicação, garantindo consistência e escalabilidade,
e utilizei o Render para hospedar o sistema, garantindo um ambiente robusto
e confiável para os usuários finais.
